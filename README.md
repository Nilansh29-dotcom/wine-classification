# wine-classification
Machine Learning model to predict wine types
#  Wine Classification

## Project Overview

This project classifies wines into different categories based on their chemical properties using Machine Learning.

##  Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
  

##  Steps Performed

* Data Loading
* Exploratory Data Analysis (EDA)
* Feature Scaling
* Model Building (KNN)
* Model Evaluation

##  Model Performance

* Accuracy: 68%
* Confusion Matrix, f1 score and precision-recall used for evaluation

## How to Run

1. Install libraries:
   pip install pandas numpy 

2. Run:
   wine_classification.ipynb

## Files

* Wine Classification.ipynb
